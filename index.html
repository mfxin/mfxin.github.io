定时器中断

发表于 2019-07-01
51单片机中共有两个16位的可编程定时器/计数器，即T0，T1.52单片机中多了一个T2. 他们既有定时功能，也有计数功能。当定时器的计数器计满后，会产生中断，及通知CPU该 如何处理。 TMOD是定时器/计数器的工作方式寄存器，确定工作方式和功能；TCON是控制寄存器， 控制T0、T1的启动和停止及设置溢出标志。 溢出时计数器的值减去计数初值才是加1计数器的计数值。 设置为定时器模式时： 加1计数器对内部机器周期计数（1个计数周期为十二个振荡周期 ，即计数频率为晶振频率的1/12）.计数值N乘以机器周期T就是定时时间t。 设置为计数器模式时： 外部事件计数脉冲由T0、T1引脚输入到计数器。在每个机器周期的S5P2期间采样T0、T1 引脚电平。当某周期采样到一高电平输入，下一周期又采样到一低电平输入，则计数器加1. 定时器的初值： THX=(65536-N)/256; TLX=(65536-N)%256; C51的中断函数格式： void 函数名() interrupt 中断号 { 中断服务程序内容; }

单片机相关电子元件介绍

发表于 2019-01-31
发光二极管LED I:3~20mA (电流只能从阳极流向阴极） U：约1.7V （导通压降） 分类： 直插式：长脚为阳极 贴片式：有标记的一端为阴极 注：不能直接连接于IO口，Vcc=5V,需要连接限流电阻（约1K）,可通过减小 电阻阻值来加强LED亮度。数码管 8位a,b,c,d,e,f,g,dp(小数点)。 共阳连接，共阴连接。 点亮数码管内部的led需要5mA以上的电流，单片机IO口送不出这么大的电流， 所以单片机与数码管连接时需要加驱动电路（可以加上拉电阻或者使用专门的 数码管驱动芯片）。 当多个数码管连接在一起时（多位数码管），它们内部的公共端（位选线）是独立 的，而负责显示什么数字的段选线是连接在一起的，独立的公共端可以控制哪个数码管亮， 段选线负责控制数码管显示什么数字。
