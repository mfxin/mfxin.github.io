定时器中断 

51单片机中共有两个16位的可编程定时器/计数器，即T0，T1.52单片机中多了一个T2.

他们既有定时功能，也有计数功能。当定时器的计数器计满后，会产生中断，及通知CPU该

如何处理。

TMOD是定时器/计数器的工作方式寄存器，确定工作方式和功能；TCON是控制寄存器，

控制T0、T1的启动和停止及设置溢出标志。

溢出时计数器的值减去计数初值才是加1计数器的计数值。

设置为定时器模式时：

加1计数器对内部机器周期计数（1个计数周期为十二个振荡周期

，即计数频率为晶振频率的1/12）.计数值N乘以机器周期T就是定时时间t。

设置为计数器模式时：

外部事件计数脉冲由T0、T1引脚输入到计数器。在每个机器周期的S5P2期间采样T0、T1

引脚电平。当某周期采样到一高电平输入，下一周期又采样到一低电平输入，则计数器加1.

定时器的初值：

THX=(65536-N)/256;

TLX=(65536-N)%256;

C51的中断函数格式：

void 函数名() interrupt 中断号
{
    中断服务程序内容;
}

单片机相关电子元件介绍

发光二极管LED
I:3~20mA (电流只能从阳极流向阴极）
U：约1.7V （导通压降）
分类：
直插式：长脚为阳极
贴片式：有标记的一端为阴极
注：不能直接连接于IO口，Vcc=5V,需要连接限流电阻（约1K）,可通过减小
电阻阻值来加强LED亮度。
数码管
8位a,b,c,d,e,f,g,dp(小数点)。
共阳连接，共阴连接。
点亮数码管内部的led需要5mA以上的电流，单片机IO口送不出这么大的电流，
所以单片机与数码管连接时需要加驱动电路（可以加上拉电阻或者使用专门的
数码管驱动芯片）。
当多个数码管连接在一起时（多位数码管），它们内部的公共端（位选线）是独立
的，而负责显示什么数字的段选线是连接在一起的，独立的公共端可以控制哪个数码管亮，
段选线负责控制数码管显示什么数字。

单片机及其引脚介绍 

    最近学习51单片机，参考了诸多硬件大佬的博客，学有所获，也想将自己学到的干货拿出来分享。
一是想通过这种方式激励自己继续进行下去，另外也希望看到这篇博文的你也能学有所获。好了废话不
多说，进入正题。

1.单片机结构
    要学习单片机首先要知道单片机是什么。一台计算机主要有CPU（进行运算，控制）、RAM（数据存储-内
存）、ROM（程序存储）、I/O（串行口，并行输出口）。将这些部分做到一块集成电路芯片中去就是一
个单片机了。单片机是一种控制芯片，一个微型计算机，而加上晶振、存储器、地址锁存器、逻辑门、七
段译码器（显示器）、按钮（类似键盘）、扩展芯片、接口等便构成了单片机系统。单片机体积不大，一
般40引脚封装，功能多的引脚也比较多（68脚），功能少的有10脚、20脚，甚至8脚。

2.MCS51单片机与8051，8031，89C51等的关系
     MCS51单片机是指由美国INTEL生产的一系列单片机的总称，其中包括8031，8751，8032，8052，8752等。
我学单片机是从89C51开始的。

3.单片机引脚介绍
单片机的40个引脚大致可以分为四类：电源、时钟、控制、I/O引脚。
（1）电源
VCC  芯片电源，接+5V；
VSS  接地端；
（2）外接晶振引脚
XTAL1(19脚）、XTAL2（18脚）：接石英晶体振荡器；
（3）控制引脚
RST（19脚）：复位信号引脚。当石英晶体振荡器运行时，在此引脚上出现的两个机器周期的高电平，使单片机
复位。
ALE/PROG（30脚）：地址锁存允许信号端。它可以用做对外输出的时钟或用于计时。
PSEN（29脚）：外部程序存储器的读选通信号端。
EA/VPP（31脚）：访问程序存储器选择控制信号。当EA信号接低电平时，对ROM的读操作限定在外部存储器；当
EA信号接高电平时，对ROM的读操作从内部开始。

4.I/O引脚（32个）
80C51共有4个8位输入/输出端口，分别为P0,P1,P2,P3。每个端口有32个引脚，总计32个引脚，每个引脚都可单
独作为I/O使用。其中P3端口还存在第二功能。
P3.0    RXD    串行数据接收端
P3.1    TXD    串行数据发送端
P3.2    INT0   外部中断0申请信号线
P3.3    INT1   外部中断1申请信号线
P3.4    T0       定时器/计数器0计数输入端
P3.5    T1       定时器/计数器1计数输入端
P3.6    WR     外部数据存储器写选通端
P3.7    RD      外部数据存储器读选通端
